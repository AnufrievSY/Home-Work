1.  Подключить репозиторий с nginx любым удобным способом, установить nginx и потом удалить nginx,
    используя утилиту dpkg.

    student@Ubuntu-MySQL-VirtualBox:~$ echo "deb http://nginx.org/packages/ubuntu focal nginx" > nginx.list
    student@Ubuntu-MySQL-VirtualBox:~$ sudo apt install code-aster-mpi-engine
    student@Ubuntu-MySQL-VirtualBox:~$ curl -fsSL https://nginx.org/keys/nginx_signing.key | sudo apt-key add -
        OK
    student@Ubuntu-MySQL-VirtualBox:~$ nginx -V
        nginx version: nginx/1.10.3 (Ubuntu)

    student@Ubuntu-MySQL-VirtualBox:~$ sudo apt update
    student@Ubuntu-MySQL-VirtualBox:~$ sudo apt install nginx -y


    student@Ubuntu-MySQL-VirtualBox:~$ curl -fsSL https://nginx.org/keys/nginx_signing.key | sudo apt-key add -
    OK

    student@Ubuntu-MySQL-VirtualBox:~$ dpkg --get-selections|grep nginx
    nginx						install
    nginx-common					install
    nginx-core					install

    student@Ubuntu-MySQL-VirtualBox:~$ sudo apt-get --purge remove nginx
        Следующие пакеты будут УДАЛЕНЫ:
          nginx*
        Удаляется nginx (1.10.3-0ubuntu0.16.04.5) …

    student@Ubuntu-MySQL-VirtualBox:~$ dpkg --get-selections|grep nginx
    nginx-common					install
    nginx-core					install

3.  Настроить iptables: разрешить подключения только на 22-й и 80-й порты.

    student@Ubuntu-MySQL-VirtualBox:~$ sudo iptables -A INPUT -p tcp --dport 80 -j ACCEPT
    student@Ubuntu-MySQL-VirtualBox:~$ sudo iptables -A INPUT -p tcp --dport 20 -j ACCEPT

    student@Ubuntu-MySQL-VirtualBox:~$ sudo iptables-save
        # Generated by iptables-save v1.6.0 on Sun Feb 26 11:41:50 2023

        -A INPUT -p tcp -m tcp --dport 20 -j ACCEPT
